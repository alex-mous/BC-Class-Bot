Welcome to the Tech Club Bot!

Introduction

This is the webserver for the Bellevue College Tech Club Discord Bot. Join the bot development server here: https://discord.gg/cF7R349 and see the Bot's GitHub page here: https://github.com/polarpiberry/TechClubBot

Installation

To add the Bot to your Discord server, please contact me through the GitHub page for the OAuth access.

Usage

Access the follow bot help page with !help. All voting commands, as well as those under the Admin Only heading require the "Admin" role.

General Command

Command Description
hi Greetings from the bot
say Speak up, little one!
status My status
help Show the help
Admins Only
vote Create a vote/poll of the channel (enter Voting Mode)
selfdestruct Self destruct the channel
ban @USER Ban @USER
deleteall Delete all messages in channel

Voting Mode Commands

Command Description
Admins Only
start Option 1|Option 2|...|Option N Start the vote with the various options
cancel Cancel the vote before it starts
help Show the help
